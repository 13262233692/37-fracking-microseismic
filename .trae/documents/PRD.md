## 1. 产品概述

水力压裂微震实时反演系统——页岩气储层工程的硬核武器，用于实时反演千米地下水力压裂激活的裂缝网络几何拓扑。系统打通从微震波形流摄取、自动震相拾取到三维走时层析成像反演与等值面实时渲染的全链路，为压裂工程师提供裂缝扩展的时空演化可视化决策支撑。

- 目标用户：页岩气田压裂工程师、地球物理研究员、储层监测团队
- 核心价值：将地下千米级压裂缝网的时空演化从"黑箱"变为"透明"，支撑实时压裂参数优化与诱震风险评估

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 压裂工程师 | 管理员分配账号 | 实时监测压裂缝网扩展、调整滤波参数、启动/停止反演 |
| 地球物理研究员 | 管理员分配账号 | 查看历史反演结果、导出数据、调整层析成像参数 |
| 系统管理员 | 系统内置 | 管理台站配置、用户管理、系统运维 |

### 2.2 功能模块

1. **实时监测仪表盘**：系统总览、台站状态、波形瀑布流、实时微震事件计数
2. **波形处理工作台**：多台站波形显示、滤波器参数调节、STA/LTA 拾取结果可视化
3. **层析成像反演**：三维速度结构等值面渲染、走时残差分析、反演迭代收敛监控
4. **事件目录与回放**：历史微震事件检索、波形回放、反演结果对比

## 3. 核心流程

用户打开系统后，首先在实时监测仪表盘查看台站运行状态与波形流。当检测到微震事件簇后，可在波形处理工作台调节滤波参数并验证 STA/LTA 拾取质量。确认到时可靠后，启动层析成像反演，系统在后台并行运行 LSQR 求解器，迭代过程中实时推送收敛曲线与中间等值面到前端。反演完成后，三维等值面完整渲染呈现裂缝扩展范围。

    "巴特沃斯带通滤波" --> "STA/LTA 自动拾取"
    "STA/LTA 自动拾取" --> "P 波到时序列"
    "P 波到时序列" --> "走时残差矩阵构建"
    "走时残差矩阵构建" --> "LSQR 层析反演"

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深空黑 (#0A0E17) + 钢铁灰 (#1A1F2E)，体现地下工程的深沉与专业
- 强调色：岩浆红 (#FF3D00) 用于关键告警与活跃裂缝，电弧蓝 (#00E5FF) 用于数据流与拾取标注，矿脉金 (#FFD600) 用于反演结果高亮
- 按钮风格：圆角矩形，微发光边框，悬停时发光增强
- 字体：显示字体 JetBrains Mono（代码/数据感），正文 IBM Plex Sans（工业科技感）
- 布局风格：深色仪表盘式，左侧导航 + 顶部状态栏 + 主内容区网格布局
- 图标风格：线性描边图标，与数据可视化风格统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 实时监测仪表盘 | 台站状态卡片 | 深色卡片，LED 指示灯（绿/红），采样率数字，脉冲动画 |
| 实时监测仪表盘 | 波形瀑布流 | 深色背景，绿色/蓝色波形线，垂直时间轴滚动 |
| 实时监测仪表盘 | 微震事件计数器 | 折线趋势图，数字翻牌器，脉冲指示 |

### 4.3 响应式设计

- 桌面优先设计，主分辨率 1920×1080 及以上
- 波形与三维视图最小宽度 1024px，低于此宽度隐藏侧栏
- 支持双屏布局：主屏三维渲染，副屏波形与参数

### 4.4 3D 场景指引

- 环境氛围：深色地下空间感，微弱环境光
- 光照：双光源（暖色主光 + 冷色补光），增强等值面立体感
- 相机：默认俯视 45° 角度，支持自由旋转/缩放/平移
- 交互：点击等值面弹出局部波速信息，拖拽旋转，滚轮缩放
- 后处理：边缘发光（bloom）效果突出异常区域
